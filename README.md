# Namma HomeStay

Namma HomeStay is an Android application developed for homestay booking and hospitality management. The application helps users explore homestays, view accommodation details, and perform booking operations through a simple mobile interface.

The project was developed using Kotlin and Android Studio with Firebase Authentication and Firebase Realtime Database integration for secure login and booking management.

## Problem Statement

Many rural and local homestay owners face difficulties in promoting their accommodations digitally due to lack of simple booking systems and technical knowledge. Travelers also find it difficult to access authentic local homestay information easily.

This project provides a user-friendly platform for viewing homestays, checking accommodation details, and connecting travelers with homestay owners digitally.

## Features

• User Registration and Login  
• Homestay Listings  
• Booking Management  
• Firebase Realtime Database Integration  
• AI-Based Travel Assistance  
• Contact Host Feature  
• Responsive User Interface  

## Technologies Used

• Kotlin  
• Android Studio  
• Firebase Authentication  
• Firebase Realtime Database  
• XML UI Design  
• RecyclerView  
• CardView  
• Gemini API  

## Installation

1. Clone the repository  
2. Open the project in Android Studio  
3. Sync Gradle dependencies  
4. Run the application on an emulator or Android device
